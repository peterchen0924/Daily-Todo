本週學習：
真正的自由，不是「選擇談」或「選擇不談」。
真正的自由是：你清楚知道自己為什麼做這個選擇，而且無論選擇哪一個，你的內在平安都不會因此動搖。
不確定，也是一種很好的狀態。它代表你沒有被恐懼或社會時鐘綁架，你正在用自己的節奏，走一條屬於你的路。

本週目標：

第一優先（firmware 直接相關）

位運算：比特位計數、只出現一次的數字、或運算最小翻轉次數
鏈表：反轉鏈表、刪除鏈表中間節點、奇偶鏈表
哈希表：找出兩數組的不同、獨一無二的出現次數

第二優先（基礎必會）

雙指針：移動零、判斷子序列
滑動窗口：子數組最大平均數 I
數組/字符串的 Easy 題全部

圖論（DFS/BFS）、動態規劃、前綴樹、區間集合 → 對 firmware 面試幫助有限，留到後期
